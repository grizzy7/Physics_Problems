### Finding Local Maxima or Minima

Given the function:

f(x) = 3x² − 12x + 7

### Step 1: Find the first derivative

f'(x) = d/dx (3x² − 12x + 7)

f'(x) = 6x − 12

### Step 2: Find critical points

Set the derivative equal to zero:

6x − 12 = 0

6x = 12

x = 2

### Step 3: Determine if it is a maximum or minimum

Find the second derivative:

f''(x) = 6

Since **f''(x) > 0**, the function is **concave up**, so the point is a **local minimum**.

### Step 4: Find the y-value

f(2) = 3(2²) − 12(2) + 7

f(2) = 3(4) − 24 + 7

f(2) = 12 − 24 + 7

f(2) = −5

### Final Result

Local minimum at:

(2, −5)

There is **no local maximum** because the parabola opens upward.
