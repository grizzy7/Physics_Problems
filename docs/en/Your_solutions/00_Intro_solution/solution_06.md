To find the **components of the vector**, we use:

* Horizontal component: (A_x = A\cos\theta)
* Vertical component: (A_y = A\sin\theta)

Given:

* Magnitude (A = 15)
* Angle (\theta = 60^\circ)

### 1. Horizontal component

[
A_x = 15\cos(60^\circ)
]

[
\cos(60^\circ) = 0.5
]

[
A_x = 15 \times 0.5 = 7.5
]

### 2. Vertical component

[
A_y = 15\sin(60^\circ)
]

[
\sin(60^\circ) \approx 0.866
]

[
A_y = 15 \times 0.866 \approx 12.99
]

### ✅ Final Answer

* **Horizontal component:** (A_x = 7.5)
* **Vertical component:** (A_y \approx 12.99) (≈ 13)

📌 So the vector components are approximately:

[
(7.5,; 13)
]
